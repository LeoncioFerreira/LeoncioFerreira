## 💡 Sobre mim

Desenvolvedor backend de **Juazeiro do Norte/CE**, focado em APIs robustas, arquitetura de software e construção de soluções que resolvem problemas reais.

Acredito em engenharia guiada por boas práticas, aplicando **TDD**, **SOLID** e **POO** para construir sistemas confiáveis e sustentáveis.

No desenvolvimento backend, vejo testes não como opcionais, mas como **obrigatórios**.

Uma única mudança pode gerar efeitos em cascata por todo o sistema, e testes automatizados são fundamentais para garantir confiabilidade, prevenir regressões e sustentar a evolução do software.

Para mim, testes são parte essencial de qualquer aplicação backend de sucesso.

Tenho interesse em **arquitetura limpa**, **performance**, **sistemas distribuídos** e desenvolvimento de produtos com impacto real.
